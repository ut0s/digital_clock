# Digital Clock for Streaming

A simple, full-screen digital clock built with Nuxt 3.
Displays time in `HH:MM:SS` format using a clean monospace font.

## Features

- Full-screen display
- Responsive font size (`18vw`)
- Modern "JetBrains Mono" font
- Dark theme with subtle gradient (optimized for streaming)

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Production

```bash
# Build the application
npm run build

# Preview production build
npm run preview
```
